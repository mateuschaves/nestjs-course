## Description

Task management API developed in NestJS Zero to Hero course

## Installation

```bash
$ npm install
```

## Functionalities

- [x] create a task
- [x] list all task
- [x] show a task
- [x] filter tasks by status, description and title
- [x] update a task
- [x] delete a task

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
